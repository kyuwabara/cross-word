# クロスワードパズル アプリケーション

5歳児のひらがな学習用クロスワードパズル生成 Web アプリ。

## 技術スタック

- **バックエンド**: Python + Flask
- **フロントエンド**: バニラ HTML / CSS / JavaScript（フレームワーク不使用）
- **クロスワード生成**: Python によるグリッディ配置アルゴリズム

## ファイル構成

- `app.py` - Flask サーバー（API エンドポイント）
- `crossword.py` - クロスワード生成ロジック
- `words.py` - 5歳児向け単語リスト（ひらがな + ヒント）
- `templates/index.html` - HTML テンプレート
- `static/style.css` - スタイルシート
- `static/script.js` - クライアント側ロジック

## 起動方法

```bash
python3 -m venv venv
source venv/bin/activate
pip install flask
python app.py
```

`http://<ホストのIPアドレス>:8080` にタブレットからアクセス。

## 仕様

### クロスワードルール
- ひらがなのみ使用
- 小さい仮名（ゃゅょっ）は大きい仮名（やゆよつ）に変換して配置（日本語クロスワードの慣例）
- 各単語は2文字以上
- 最初の単語以外は既存の単語と必ず交差
- 単語の前後は黒マスまたは端
- 意図しない並行単語の生成を防止

### 画面仕様
- タブレット操作に最適化（タッチ対応、大きなマス・ボタン）
- ひらがなオンスクリーンキーボード（清音・濁音・半濁音）
- セルタップで選択、再タップで縦横切り替え
- 「こたえあわせ」で正誤を色表示
- グリッドサイズ 5×5 〜 15×15 で指定可能
