# クロスワードパズル

5歳児のひらがな学習用クロスワードパズルを自動生成する Web アプリケーションです。

PWA（Progressive Web App）としてブラウザ単体で動作します。初回にオンラインで読み込めば、以降はオフラインで利用できます。

## 利用方法

https://kyuwabara.github.io/cross-word/ にアクセスするだけです。

### タブレットでオフラインで使う

1. オンライン状態で上記 URL にアクセス（Service Worker が自動でキャッシュします）
2. Chrome のメニューから **「ホーム画面に追加」** で PWA 化
3. 以降は機内モードでも、ホーム画面のアイコンから起動して遊べます

## ローカル開発

静的ファイルを配信するだけなので、任意の HTTP サーバーで動きます。

```bash
python3 -m http.server 8080
```

ブラウザから `http://localhost:8080/` にアクセスしてください。

## 機能

### パズル生成
- グリッドサイズを 5×5 〜 15×15 で指定可能
- 340 語超の単語リストからランダムに選択・配置
- ページを開くと自動でパズルが生成される

### 操作
- セルをタップして選択、もう一度タップで縦横方向を切り替え
- 選択中の単語全体がハイライトされ、対応するヒントも強調表示される
- 五十音表レイアウト（清音・濁音・半濁音）のオンスクリーンキーボードで入力
- キーボードの「けす」ボタンで 1 文字削除、「ぜんぶけす」で全リセット
- ヒントをタップすると対応するセルの先頭にジャンプ

### こたえあわせ
- 正解のセルは緑、不正解のセルは赤で表示
- 不正解のセルには正しい文字が青字で表示される
- 全問正解・部分正解のメッセージが出る

### 印刷モード
- 「いんさつ」ボタンで空のパズル（グリッド＋ヒント）を印刷可能
- 紙に書いてひらがなの練習に使える

## クロスワードのルール

- ひらがなのみ使用（簡単な外来語もひらがな表記）
- 小さい仮名（ゃゅょっ）は大きい仮名（やゆよつ）に変換して配置（日本語クロスワードの慣例）
- 各単語は2文字以上
- すべての単語は他の単語と交差する
- 単語の前後は黒マスまたはグリッドの端

## 技術スタック

- **フロントエンド**: HTML / CSS / JavaScript（フレームワーク不使用）
- **オフライン対応**: Service Worker（cache-first）+ Web App Manifest
- **ホスティング**: GitHub Pages（静的ファイルのみ）
- **クロスワード生成**: 貪欲配置アルゴリズム（複数回試行して最良の結果を採用）

## ファイル構成

```
├── index.html              # エントリポイント
├── manifest.webmanifest    # PWA マニフェスト
├── sw.js                   # Service Worker
└── static/
    ├── style.css           # スタイルシート
    ├── script.js           # UI ロジック
    ├── crossword.js        # クロスワード生成アルゴリズム
    ├── words.js            # 単語リスト（ひらがな + ヒント）
    └── icon.svg            # PWA アイコン
```
