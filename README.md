# クロスワードパズル

5歳児のひらがな学習用クロスワードパズルを自動生成する Web アプリケーションです。

ノート PC でサーバーを起動し、家庭内 LAN でタブレットからアクセスして使用します。

## セットアップ

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## 起動方法

```bash
source venv/bin/activate
python app.py
```

タブレットのブラウザから `http://<PC の IP アドレス>:8080` にアクセスしてください。

IP アドレスはサーバー起動時のログに表示されます。

## 機能

### パズル生成
- グリッドサイズを 5×5 〜 15×15 で指定可能
- 217語の単語リストからランダムに選択・配置
- ページを開くと自動でパズルが生成される

### 操作
- セルをタップして選択、もう一度タップで縦横方向を切り替え
- 五十音表レイアウトのオンスクリーンキーボードで入力
- ヒントをタップすると対応するセルにジャンプ

### こたえあわせ
- 正解のセルは緑、不正解のセルは赤で表示
- 不正解のセルには正しい文字が青字で表示される

### 印刷モード
- 「いんさつ」ボタンで空のパズル（グリッド＋ヒント）を印刷可能
- 紙に書いてひらがなの練習に使える

## クロスワードのルール

- ひらがなのみ使用（簡単な外来語もひらがな表記）
- 小さい仮名（ゃゅょっ）は大きい仮名（やゆよつ）に変換して配置（日本語クロスワードの慣例）
- 各単語は2文字以上
- すべての単語は他の単語と交差する
- 単語の前後は黒マスまたはグリッドの端

## 技術スタック

- **バックエンド**: Python / Flask
- **フロントエンド**: HTML / CSS / JavaScript（フレームワーク不使用）
- **クロスワード生成**: 貪欲配置アルゴリズム（複数回試行して最良の結果を採用）

## ファイル構成

```
├── app.py              # Flask サーバー
├── crossword.py        # クロスワード生成アルゴリズム
├── words.py            # 単語リスト（217語 + ヒント）
├── templates/
│   └── index.html      # HTML テンプレート
├── static/
│   ├── style.css       # スタイルシート
│   └── script.js       # クライアント側ロジック
└── requirements.txt    # Python 依存パッケージ
```
